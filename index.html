The International Standard for the Verification of Neural Networks

Promoting greater interoperability and rigour in the formal verification of neural networks.

The VNN-LIB Standard provides syntax and semantics to express satisfiability queries over neural networks, and a command-line interface for neural network solvers that answer these queries. See the latest standard document.

About

The VNN-LIB standard is a format that is expressive, and rigorous to express neural network verification tasks. It also standardizes the interface to solvers, and formalizes the semantics of this format. Notable improvements of the tailored format include:

  • The ability to express multiple networks, inputs, outputs, and hidden nodes.
  • The reduced verbosity, and increased interpretability of the format.
  • The formalization of this format.
(vnnlib-version <2.0>) ; Network declaration (declare-network f (declare-input X float32 [2,2]) (declare-output Y float32 [1]) ) ; Input bounds (assert (and (>= X[1,0] 0.0) (<= X[0,0] 1.0))) (assert (and (>= X[0,1] 0.0) (<= X[0,1] 1.0))) (assert (and (>= X[1,0] 0.0) (<= X[1,0] 1.0))) (assert (and (>= X[1,1] 0.0) (<= X[1,1] 1.0))) ; Output bounds (assert (or (<= Y[0] -3.0) (>= Y[0] 0.0)))

Libraries

A selection of official libraries and packages related to the VNN-LIB standard.

C++

parser

A C++ library for parsing VNN-LIB files. It contains a main parsing function for generating an AST from a VNN-LIB query, and utilities to transform the AST into other formats.

Python

parser

A Python package for parsing VNN-LIB files. This package implements python bindings for the C++ VNN-LIB parser.

Julia

parser

A Julia package for parsing VNN-LIB files. This package wraps the C++ VNN-LIB parser and exposes its functionality to Julia users.

Agda

parser

A library in the Agda theorem prover for parsing VNN-LIB files. Also contains a mechanised definition of the semantics of a subset of VNN-LIB queries.
