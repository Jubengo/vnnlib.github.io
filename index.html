The International Standard for the Verification of Neural Networks

Promoting greater interoperability and rigour in the formal verification of neural networks.

The VNN-LIB Standard provides syntax and semantics to express satisfiability queries over neural networks, and a command-line interface for neural network solvers that answer these queries. See the Standard Document.

parser
